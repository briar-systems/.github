# Briar Systems

Systems software, built from first principles.

Briar Systems is an independent software company building low-level tools,
languages, and the occasional thing that does not fit a category. The common
thread is doing things the right way over the quick way: owning the whole stack,
sweating the foundations, and building to last.

### Projects
- **[Mach](https://machlang.org)** is a low-level, explicitly-typed programming
  language and self-contained toolchain, with its own compiler backend, linker,
  and standard library, and no LLVM.

More to come.
